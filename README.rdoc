= Iruinai

その日既にログインしたプロジェクトメンバーの数を表示します。
とりあえずチケット画面のサイドバーにのみ表示するようにしています。


== インストール

1. RAILS_ROOT/vendor/pluginsに配置する。

2. Redmineを再起動する。


== アンインストール

1. RAILS_ROOT/vendor/pluginsから削除する。

2. Redmineを再起動する。
